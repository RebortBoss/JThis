JThis -- Java Web 开发框架
------

**JThis** 是一款通用的Java Web开发框架，使用Spring + Mybatis + Druid + FastJson等技术，可以作为**移动App的后台接口服务开发框架**


#### 基本功能：
> * URL支持Restful
> * Druid实时监控
> * Spring MultiFile多文件上传
> * FastJson解析
